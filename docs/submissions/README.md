# Submissions

Papers and artefacts SCPN-MIF-CORE submits or prepares for submission to external
venues. Sources MIF *cites* live in `docs/external_sources/`; this directory holds
the works MIF *authors*.

| Submission | Path | Venue | Status |
|---|---|---|---|

## Notes

- The JOSS paper draft PDF is built by the `JOSS paper draft` workflow on changes
  under `docs/submissions/`.
- References must be registered in `docs/external_sources/` and verified at source
  before they enter a submission's bibliography.
